Instruções de Configuração (Windows)

1. Navegar até o diretório

cd C:\Users\Guilherme\Desktop\rdo

2. Criar ambiente virtual

python -m venv venv

3. Ativar ambiente virtual

.\venv\Scripts\activate

4. Instalar dependências

pip install -r requirements.txt

5. Rodar aplicação

flask run

6. Desativar ambiente (ao terminar)

deactivate

Notas importantes:

  • Certifique-se que o XAMPP está rodando
  • Acesse: http://localhost:5000
  • Para voltar a trabalhar depois, execute os passos 1 e 3
